# CENO HTT PolyQ Delta Scoring Pipeline

This repository contains a minimal, auditable pipeline for finetuning the
[CENO](https://huggingface.co/CladeTeam) DNA language model on CDS sequences
and scoring HTT/polyQ Q-sweep sequences.

The main reported value is:

```text
delta(Q) = avg_log_likelihood(finetuned_model, sequence_Q)
         - avg_log_likelihood(pretrained_model, sequence_Q)
```

For transparency, the scoring scripts also export the exact DNA sequence used at
each Q count and the raw pretrained/finetuned scores.

## CENO Models

The CENO model is derived from NVIDIA's Nemotron-H (Apache-2.0). Pretrained and
species-specific CDS-finetuned checkpoints are published on Hugging Face under
the `CladeTeam` organization:

| Model | repo_id |
|-------|---------|
| Base (preview) | `CladeTeam/CENO-base-1b-preview` |
| Chlamydomonas CDS finetune | `CladeTeam/CENO-chlamydomonas-cds` |
| Rice CDS finetune | `CladeTeam/CENO-rice-cds` |

Each model repository bundles the custom Transformers remote code
(`configuration_ceno.py`, `modeling_ceno.py`, `ceno_tokenizer.py`) alongside the
weights, so a single snapshot download gives you both the model and the code
needed to load it.

## Repository Layout

```text
open_source_delta_pipeline/
  htt_delta/                 # importable, tested Python package
    sequences.py             # HTT/polyQ sequence definitions
    fasta.py                 # FASTA readers/writers
    model_io.py              # local CENO loading
    scoring.py               # avg log-likelihood scoring
  scripts/                   # command-line entry points
    prepare_cds_data.py
    finetune_cds.py
    export_sequences.py
    score_delta.py
    plot_delta.py
    plot_raw_scores.py
    run_from_config.py
  configs/                   # editable JSON experiment configs
  run_examples/              # one-command example launchers
  tests/                     # lightweight sequence-boundary tests
  example_outputs/           # q_count,delta CSVs from the associated runs
```

## Supported Sequence Families

`htt_delta/sequences.py` defines the sequence families used for scoring:

- `human_htt_72q_natural_trunc`
  - Human HTT 72Q natural CAG/CAA anchor.
  - The Q tract starts after `...AAGCTTC`.
  - Q=10..72 is generated by truncating codons from the polyQ end.
  - The Q=72 anchor has `CAG=36`, `CAA=36`.

- `synthetic_htt_pure_cag`
  - Synthetic human HTT exon1 sequence.
  - Fixed human HTT prefix/suffix with pure `CAG * Q`.

- `cr_htt_72q_original`
  - Chlamydomonas `Cr_HTT` 72Q-series.
  - Natural CAG/CAA layout and 72Q flanking context.
  - Q=10..72 is generated by truncating codons from the polyQ end.

## Requirements

Install:

```bash
pip install -r requirements.txt
```

You also need one or more CENO model directories. Download a model snapshot
from Hugging Face — the resulting directory contains both the weights and the
custom remote code, and is used directly as both the model path and the
model-code directory:

```bash
# Base (pretrained) model
huggingface-cli download CladeTeam/CENO-base-1b-preview \
  --local-dir models/CENO-base-1b-preview

# A species-specific finetune
huggingface-cli download CladeTeam/CENO-rice-cds \
  --local-dir models/CENO-rice-cds
```

Each downloaded directory contains `model.safetensors` plus the custom code
files (`configuration_ceno.py`, `modeling_ceno.py`, `ceno_tokenizer.py`). The
scoring scripts take this directory as `--pretrained-model` / `--finetuned-model`
(weights + config) and `--model-code-dir` (the code files); both can point at
the same downloaded directory.

You also need CDS FASTA files if you want to reproduce finetuning.

## Prepare CDS Data

```bash
python scripts/prepare_cds_data.py \
  --input-cds path/to/species.cds.fa \
  --output-dir finetune_data/rice \
  --train-frac 0.9 \
  --seed 42
```

## Finetune

Use `torchrun` for multi-GPU finetuning:

```bash
torchrun --nproc_per_node=8 scripts/finetune_cds.py \
  --model-path models/CENO-base-1b-preview \
  --model-code-dir models/CENO-base-1b-preview \
  --train-fasta finetune_data/rice/train.fa \
  --val-fasta finetune_data/rice/val.fa \
  --output-dir finetuned_rice \
  --epochs 1 \
  --batch-size 1 \
  --grad-accum 8 \
  --max-length 8192
```

The best model is saved to:

```text
finetuned_rice/best_model
```

## Score From A Config

Edit one of the JSON configs under `configs/` to point at your local model
directories (the snapshot downloads from the previous step). For example,
`configs/rice_human_htt_72q_natural_trunc.json`:

```json
{
  "pretrained_model": "models/CENO-base-1b-preview",
  "finetuned_model": "models/CENO-rice-cds",
  "model_code_dir": "models/CENO-base-1b-preview",
  ...
}
```

`model_code_dir` can point at any downloaded CENO directory (base or finetuned)
since the bundled code files are identical across them. Then run:

```bash
python scripts/run_from_config.py \
  --config configs/rice_human_htt_72q_natural_trunc.json \
  --gpu 0
```

This writes:

- `*_q_delta.csv`: two-column output, `q_count,delta`
- `*_scores.csv`: raw pretrained/finetuned scores and sequence metadata
- `*_sequences.fa`: exact DNA sequence scored for each Q
- `*_delta.png`: delta plot
- `*_raw_scores.png`: raw score plot

## Score Manually

```bash
python scripts/score_delta.py \
  --sequence human_htt_72q_natural_trunc \
  --pretrained-model models/CENO-base-1b-preview \
  --finetuned-model models/CENO-rice-cds \
  --model-code-dir models/CENO-base-1b-preview \
  --output outputs/rice_human_htt_72q_natural_trunc_q_delta.csv \
  --details-output outputs/rice_human_htt_72q_natural_trunc_scores.csv \
  --sequence-output outputs/rice_human_htt_72q_natural_trunc_sequences.fa \
  --gpu 0
```

## Plot

```bash
python scripts/plot_delta.py \
```

## Example Outputs

The `example_outputs/` directory includes q/delta CSV files for:

- `synthetic_htt_pure_cag` scored with a rice-finetuned model

```text
q_count,delta
```

## Tests

Run the lightweight sequence-boundary tests:

```bash
python -m pytest tests
```

These tests verify Q-count boundaries, anchor lengths, and CAG/CAA counts. They
do not require model weights or a GPU.
